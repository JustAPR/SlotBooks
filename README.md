
# Slot Booking

A mock slot booking application in ReactJS




## Deployment

To deploy this project run

```bash
  cd backend
  npm install
  node server.js
```
New terminal in same dir
```bash
  cd frontend
  npm install
  npm start
```

## Environment Variables
